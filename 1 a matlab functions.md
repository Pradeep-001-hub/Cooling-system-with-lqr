 MATLAB Functions Used

This project utilizes a wide range of MATLAB Control System Toolbox functions:

Modeling:
ss, tf, zpk, ss2tf, tf2ss

Simulation:
step, impulse, lsim, initial

Controllers:
pid, lqr, lqe, kalman, lqg

Analysis:
eig, pole, zero, damp, isstable

System Properties:
ctrb, obsv, rank

Frequency Response:
bode, nyquist, margin

Performance Evaluation:
stepinfo

Plotting:
plot, subplot, figure, title, xlabel, ylabel, legend, grid


Key Concepts

- State-Space Modeling
- PID Control
- Linear Quadratic Regulator (LQR)
- Linear Quadratic Gaussian (LQG)
- Kalman Filtering
- Stability & Performance Analysis

 File Types

- ".m" MATLAB scripts
- ".slx" Simulink models
- ".png" Output graphs
- "README.md" Documentation
