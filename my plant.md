System Architecture

The proposed system consists of the following components:

1. Plant (Thermal System)
2. LQR Controller
3. Kalman Filter (State Estimator)
4. Feedback Loop

 Working Principle

- The plant outputs temperature measurements
- The Kalman filter estimates the system states
- The LQR controller computes optimal control input
- The control input adjusts the cooling mechanism
- The loop continues to maintain desired temperature

This closed-loop structure ensures optimal and robust thermal regulation.
